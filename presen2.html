Density Estimation

The ingredients for density estimation

  • We have the Epanechnikov Kernel given as \[K(x) = \frac{3}{4}(1-x^2)1_{[-1,1]}(x)\]

  • We can then implement the density estimation of \(K(x)\) using the following formula \[\hat{f}_h(x) = \frac{1}{hn}\sum_{j=0}^{n}K(\frac{x-x_j}{h})\]

Vectorized R implementation

epan_kern_vec <- function(x, h, m = 512) {
    rg = range(x)
    xx <- seq(rg[1] - 3 * h, rg[2] + 3 * h, length.out = m)
    y <- numeric(m)
    
    const = length(x) * h
    y <- sapply(xx, function(z) sum( (abs((z-x)/h) <= 1) * (3/4 * (1 - ((z - x)/(h))^2)))) 
    list(x = xx, y = y/const)
}
  • We use sapply to map all of the gridpoints over into the density estimation
  • We use the fact that the boolean test abs((z-x)/h) <= 1) gets coerced into a numeric value

Rccp implementation

#include <Rcpp.h>
using namespace Rcpp;
using namespace std;

// [[Rcpp::export]]
NumericVector epan_den(NumericVector x, double h, int m = 512)
{
    int n = x.size();
    
    auto max = max_element(begin(x), end(x));
    auto min = min_element(begin(x), end(x));
    
    NumericVector grid(m);
    double l = abs(*max - *min);
        
    for(int i = 1; i < (m - 1); ++i) {
            grid[i] =  *min + (l / m) * i;
    }
    grid[0] = *min;
    grid[m - 1] = *max; 
    
    NumericVector y(m);
    for(int i = 0; i < m; ++i) {
        for(int j = 0; j < n; ++j) {
            double u = (grid[i] - x[j]) / h;
            if(fabs(u) <= 1.0) {
                y[i] += 3 * (1 - pow(u, 2.0)) / (4);
            }
        }
            y[i] = y[i] / (n * h);
    }
    
    return y;
}

Bandwidth selection

  • Plugin method using emperical variance and guessing using mean zero gaussian density \[h1_n = (\frac{8\sqrt{\pi} ||K||_{2}^{2}}{3\sigma_{K}^{4}})^{1/5} \sigma n^{-1/5}\] We estimate \(\sigma\) with the emperical variance.

Bandwith calculation on infrared data

First observe \[\sigma_{K}^2 = \int z^2 (\frac{3}{4}(1-z^2)) dz = \frac{1}{5}\] And \[||K||_{2}^{2} = \int (\frac{3}{4}(1-z^2))^2 dz = \frac{3}{5}\] We can now implement the bandwidth estimator

sigma <- sd(logf12) # log transformed infrared data
n <- length(logf12)
h1n <- function(n) {
    (((8 * sqrt(pi) * (3/5) ) / (3 * (1/5)^2 ))^(1/5)) * n^(-1/5) * sigma
}
h1 <- h1n(n)
h1 / sqrt(5)
## [1] 0.3070334

Silverman thumb

We substitue the emperical variance with \[\tilde{\sigma} = min\{\hat{\sigma}, IQR/1.34\}\]

sigma_tilde <- min(sd(logf12), IQR(logf12) / 1.34)
h2n <- function(n) {
    (((8 * sqrt(pi) * (3/5) ) / (3 * (1/5)^2 ))^(1/5)) * n^(-1/5) * sigma_tilde
}
h2 <- h2n(n)
h2 / sqrt(5)
## [1] 0.2413377
